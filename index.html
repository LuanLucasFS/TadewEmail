TADEW

Tadew é um novo assistente virtual criado utilizando o modelo de linguagem Llama 3.2 7B, com o intuito de servir escolas, empresas, e pessoas em geral em funções diversas. O projeto se encontra atualmente em fase de desenvolvimento, e já conta com algumas funcionalidades básicas.


Algumas de suas principais funcionálidades são:

Pesquisas
Fazer pesquisas rapidas e eficientes

Solução
Resolver problemas rapidamente

Testes
Realizar testes com eficácia

Tadew é um projeto de código aberto, e pode ser utilizado por qualquer pessoa que desejar.


Para mais informações e duvidas, acesse o repositório github oficial do projeto.
